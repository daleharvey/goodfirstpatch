Welcome to goodfirstpatch, we want to make life easier to contribute to Open Source, you should contribute for a whole bunch of reasons like X, Y, Z. This is an introduction to make it clear what the site does and how to use it.

developers

Want to contribute to an open source project but don't know which or how? Great! Add your info below and we'll add you to a registry of helpful developers maintainers can contact when they need an extra set of hands. You'll be asked to authenticate with your GitHub account, so make sure to use the same email address you use there.

maintainers

Got issues that would be perfect for someone new to open source? Great! Tag them with goodfirstpatch and add the URL of your repo below. You'll be asked to authenticate to make sure you're authorized to add the repo, then you're all set!
